【远程职位】房智汇 HouseSigma 招聘前端工程师

要求

  • vue3 + typescript
  • 由于产品需要，工程师必须有一定英文文档阅读能力。（不要求口语，日常工作交流用中文）
  • 扎实计算机基本功，设置维护自己的开发环境，使用各种团队协作工具
  • 数据结构清晰、命名规范、编写高质量代码、有模块化意识、精通调试工具，对 web 性能优化有一定的了解
  • 具有极客精神
  • 根据项目需要学习新技术，新工具
    加分项：
  • Angular ，Vue 等主流前端框架经验
  • pagespeed 性能调试

职责

  • 维护开发移动端+桌面端 web
  • 根据设计师提供的 figma 稿件实现 UI
  • 针对性能不好的程序进行优化。（性能不好可能有各种原因，需要学习研究）
  • 在 issue tracker 上更新工作进度

薪资

  • 薪资范围 12~16K
  • 工资全额付给个人，需要自行办理社保

立即申请

申请的时候请说是在remotecat.us看到的. 感谢！
